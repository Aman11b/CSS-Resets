1. sass is a type of css pre processor (less and stylus other example)
2. sass can be writeen in two form
   1. scss (regualr css with few modification)
      ```scss
        .myStyle{
        color:$myColor;
        padding:$std-padding;
        font-weight:bold;
        }
      ```
   2. Sass (written in form of Ruby)
      ```sass
      .myStyle
          color:$myColor
          padding:$std-padding
          font-weight:bold
      ```
